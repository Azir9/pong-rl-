# ataria Pong

# 1 . 环境安装

## 1.1  安装gym 环境

安装的环境要老一点

`pip install gym==0.19.0 `

`pip install atari_py==0.2.6`



## 1.2 安装 ataria 游戏 环境

安装ataria环境

`pip install gym[atari,accept-rom-license]`

## 1.3 安装 torch

`conda install pytorch==1.13.1 torchvision==0.14.1 torchaudio==0.13.1 pytorch-cuda=11.7 -c pytorch -c nvidia`
